Practica 1

El objetivo central de esta práctica es simular un flujo de trabajo real de desarrollo web utilizando Git y GitHub. No se trata solo de escribir código, sino de aprender a gestionarlo, versionarlo y publicarlo de manera profesional.

Vas a crear desde cero una página web personal (también llamada "landing page") donde serás tú el protagonista. El contenido girará en torno a tus pasatiempos o hobbies, lo que le dará un toque personal y único.

FLEXBOX

Propiedades del contenedor

  • flex-direction: Especifica como colocar los elementos flexibles. Los valores son row, column, row-reverse, column-reverse.
  • flex-wrap: Especifica si los elementos flexibles deben ajustarse o no.
  • flex-flow: Es una propiedad corta para especificar las propiedades flex-direction y flex-wrap properties.
  • justify-content: Alinea los elementos flexibles sobre el eje principal.
  • align-content: Alinea las lineas de contenido flexible (aplica cuando el contenedor tiene varias lineas). Los posibles valores son: flex-start, flex-end, space-around, space-between, space-evenly y stretch (es ultimo es el default)

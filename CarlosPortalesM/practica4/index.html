Ejemplo de Selectores

Selector Universal

Selecciona todos los elementos de la página

Selector de tipo

Selecciona todos los elementos de un tipo específico

Selector por ID

Selecciona un único elemento que tenga un ID específico.

Selector por clase

Selecciona todos los elementos que tengan una clase específica

Selector de atributo

Selecciona todos los elementos que tengan un atributo específico

Selector de lista

Selecciona elementos de manera jerárquica o anidada

Selector de descendientes

Selecciona elementos que son descendientes de un elemento específico, cualquier elemento dentro de otro

Selector hijos directos

Selecciona solo los hijos directos de un elemento específico, elementos que están inmediatamente dentro de otro

Selector hermano adyacente

Selecciona el hermano inmediato siguiente de un elemento, primer hermano inmediato

Selector hermano general

Selecciona todos los hermanos de un elemento que están después de este

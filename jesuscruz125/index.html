Practica número 3

La práctica consiste en actualizar la página de pasatiempos creada anteriormente, haciendo una versión 2 que se guardará en la organización. Para esto, se deben crear cinco archivos HTML, uno por cada pasatiempo, usando etiquetas como header, nav, main, aside y footer, para que cada página muestre la información de manera ordenada y clara.
